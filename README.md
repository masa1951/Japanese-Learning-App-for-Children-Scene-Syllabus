# Japanese-Learning-App-for-Children-Scene-Syllabus
「日本語指導が必要な児童生徒」の日本語支援アプリとして、千葉県柏市で日本語支援ボランティアとして活動する石河氏が企画し、柏市児童生徒日本語支援の会の会員である近藤氏、馬場氏、藤野氏、加藤氏らのアドバイスにより構成されています。 このアプリは「日本語指導が必要な児童生徒」が基礎的な日本語を習得することで、学校での生活場面や学習場面に早期に適応できるようにとの願いから生まれました。 「学校場面」では語彙習得を、「教科書の動詞」では体系的に文法を身に付けるように設計されています。いずれも音声入りです。
